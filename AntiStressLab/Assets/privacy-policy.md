## Политика конфиденциальности приложения «Антистресс Вайб»

Приложение «Антистресс Вайб» создано как антистресс‑песочница. Мы стремимся к минимальному сбору информации.

### 1) Какие данные собираются
Само приложение **не запрашивает** у пользователя имя, телефон, email, контакты, фото/файлы и т.п.

Однако приложение содержит **рекламные SDK** (показы рекламы). Такие SDK могут автоматически собирать **технические идентификаторы и диагностические данные**, необходимые для показа и измерения рекламы, например:
- рекламный идентификатор устройства (если доступен в ОС),
- IP‑адрес, параметры устройства/ОС,
- события показов/кликов (агрегировано),
- сведения об ошибках/сбоях.

### 2) Для чего используются данные
Данные рекламных SDK используются для:
- показа рекламы,
- предотвращения мошенничества,
- измерения эффективности рекламы (статистика показов/кликов).

### 3) Передача данных третьим лицам
Данные могут передаваться **поставщикам рекламы** (рекламным сетям) исключительно в целях, описанных выше.

### 4) Хранение данных
Приложение «Антистресс Вайб» **не хранит** персональные данные пользователя на своих серверах (серверов приложения нет).
Хранение и обработка данных рекламными SDK регулируется политиками соответствующих провайдеров.

### 5) Разрешения
Приложение использует ввод касаний. Дополнительные разрешения запрашиваются только если это требуется рекламным SDK/платформой.

### 6) Контакты
По вопросам конфиденциальности: **support@ваш-домен.ru** (замените на ваш контакт).
